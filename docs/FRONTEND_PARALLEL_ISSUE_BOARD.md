# Frontend Parallel Issue Board

Date: 2026-03-18
Scope: Overview-first SaaS workspace rollout (Navigation, Overview, Invoices, Providers, Reports, Provider health, Observability)

This board translates the planning artifacts (see `docs/FRONTEND_CONVERSION_BACKLOG.md`, `docs/FRONTEND_PARALLEL_EXECUTION_MATRIX.md`, and the recent overview-first sprint definition) into discrete, assignable GitHub issues. Each row below can be copied into GitHub issues with the provided title, acceptance criteria, owner, tests, and references so work can proceed in parallel without crossing ownership boundaries.

## Issue Creation Instructions
1. Copy the “Issue title” and paste into a new GitHub issue, prefixed with the appropriate ticket number (e.g., `[BE-101] Provider test endpoint`).
2. Fill in the issue body with the description, acceptance criteria, and testing sections listed below plus references to the design docs (`docs/Web UI Design Review.md`, `docs/Invoices Web UI IA.pdf`, `docs/Invoices Web UI IA.jam`).
3. Assign the issue to the named agent, add the `wave5-8` label, and link to relevant owner docs.
4. Include at least one dependent design doc/contract reference per issue (use the `doc(s)` column).
5. On issue completion, remind the assignee to run `scripts/validate_pr.sh ... --strict` as described in `docs/qa/PR_VALIDATION_PROCESS.md`.

## Enforcing PR Discipline
- Every PR must mention the linked issues with closing keywords (e.g., `Closes #BE-101`). Use the existing PR template in `docs/qa/PR_VALIDATION_REQUIREMENTS_TEMPLATE.md`.
- Require `scripts/validate_pr.sh` with the `--strict` flag; include `--design-docs` covering the pieces touched (Overview, Providers, Reports, etc.).
- Validation artifacts (e.g., `docs/qa/PR_XX_VALIDATION_REPORT.md`) must be uploaded to the repo before merging.
- Use `docs/qa/AGENT_NEXT_ISSUE_PROMPT_TEMPLATE.md` to drive agent prompts so branch naming, validation, and documentation steps stay consistent.
- QA (Apollo) owns monitoring the validation reports and gating merges when any failures remain.
